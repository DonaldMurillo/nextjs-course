# Chapter 10.2: Font Optimization

## Overview

Fonts can significantly impact performance and user experience. Next.js provides `next/font` to automatically optimize fonts, eliminate layout shift, and improve privacy by self-hosting Google Fonts.

## Why Font Optimization Matters

Without optimization:
1. **Layout Shift (CLS)**: Text jumps when font loads
2. **Flash of Unstyled Text (FOUT)**: System font shows, then switches
3. **Privacy Concerns**: Google Fonts sends user data to Google
4. **Extra Network Requests**: Fonts loaded from external CDN

With `next/font`:
1. Fonts are self-hosted (no external requests)
2. Zero layout shift with automatic `size-adjust`
3. No tracking by font providers
4. Fonts are preloaded and optimized

## Google Fonts

### Basic Usage

```tsx
// app/layout.tsx
import { Inter } from 'next/font/google';

const inter = Inter({
  subsets: ['latin'],
});

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body className={inter.className}>{children}</body>
    </html>
  );
}
```

### With CSS Variables

CSS variables let you use fonts anywhere in CSS:

```tsx
// app/layout.tsx
import { Inter, Roboto_Mono } from 'next/font/google';

const inter = Inter({
  subsets: ['latin'],
  variable: '--font-inter',
});

const robotoMono = Roboto_Mono({
  subsets: ['latin'],
  variable: '--font-roboto-mono',
});

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en" className={`${inter.variable} ${robotoMono.variable}`}>
      <body className={inter.className}>{children}</body>
    </html>
  );
}
```

Use in CSS:

```css
/* globals.css */
h1, h2, h3 {
  font-family: var(--font-inter);
}

code, pre {
  font-family: var(--font-roboto-mono);
}
```

### Font Options

```tsx
import { Roboto } from 'next/font/google';

const roboto = Roboto({
  weight: ['400', '500', '700'],    // Specific weights
  style: ['normal', 'italic'],      // Include italic
  subsets: ['latin', 'latin-ext'],  // Character subsets
  display: 'swap',                  // Font display strategy
  variable: '--font-roboto',        // CSS variable name
  preload: true,                    // Preload font (default true)
  fallback: ['system-ui', 'arial'], // Fallback fonts
});
```

### Variable Fonts

Variable fonts support a range of weights in a single file:

```tsx
import { Inter } from 'next/font/google';

const inter = Inter({
  subsets: ['latin'],
  // Variable fonts automatically support all weights
  // No need to specify weight
});

// Use any weight in CSS
// font-weight: 100 to 900
```

## Local Fonts

### Basic Local Font

```tsx
// app/layout.tsx
import localFont from 'next/font/local';

const myFont = localFont({
  src: './fonts/MyFont.woff2',
  display: 'swap',
});

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body className={myFont.className}>{children}</body>
    </html>
  );
}
```

### Multiple Font Weights

```tsx

const myFont = localFont({

    {
      path: './fonts/MyFont-Medium.woff2',
      weight: '500',
      style: 'normal',
    },
    {
      path: './fonts/MyFont-Bold.woff2',
      weight: '700',
      style: 'normal',

    {
      path: './fonts/MyFont-Italic.woff2',
      weight: '400',
      style: 'italic',

```

## Using with Tailwind CSS

### Configure Tailwind

```ts
// tailwind.config.ts
import type { Config } from 'tailwindcss';

const config: Config = {
  content: ['./app/**/*.{js,ts,jsx,tsx}', './components/**/*.{js,ts,jsx,tsx}'],
  theme: {
    extend: {
      fontFamily: {
        sans: ['var(--font-inter)', 'system-ui', 'sans-serif'],
        mono: ['var(--font-roboto-mono)', 'monospace'],
        display: ['var(--font-display)', 'sans-serif'],
      },
    },
  },
  plugins: [],
};

export default config;
```

### Use in Components

```tsx
<p className="font-sans">Using Inter</p>
<code className="font-mono">Using Roboto Mono</code>
<h1 className="font-display">Using Display Font</h1>
```

### Complex Example: Brand Font System

```tsx
// lib/fonts.ts
import { Inter, Playfair_Display } from 'next/font/google';
import localFont from 'next/font/local';

export const inter = Inter({
  subsets: ['latin'],
  variable: '--font-inter',
});

export const playfair = Playfair_Display({
  subsets: ['latin'],
  variable: '--font-playfair',
});

export const brandFont = localFont({
  src: [
    {
      path: '../public/fonts/Brand-Light.woff2',
      weight: '300',
      style: 'normal',
    },
    {
      path: '../public/fonts/Brand-Regular.woff2',
      weight: '400',
      style: 'normal',
    },
    {
      path: '../public/fonts/Brand-Bold.woff2',
      weight: '700',
      style: 'normal',
    },
  ],
  variable: '--font-brand',
  display: 'swap',
});

export const fontVariables = `${inter.variable} ${playfair.variable} ${brandFont.variable}`;
```

```tsx
// app/layout.tsx
import { fontVariables, inter } from '@/lib/fonts';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en" className={fontVariables}>
      <body className={inter.className}>{children}</body>
    </html>
  );
}
```

## Font Display Strategies

```tsx
const font = Inter({
  subsets: ['latin'],
  display: 'swap', // Options: 'auto' | 'block' | 'swap' | 'fallback' | 'optional'
});
```

| Value | Behavior |
|-------|----------|
| `swap` | Show fallback immediately, swap when loaded (recommended) |
| `block` | Hide text briefly, then show custom font |
| `fallback` | Short block period, swap if loaded quickly |
| `optional` | Short block period, may not swap (for slow connections) |
| `auto` | Browser default |

## Key Takeaways

- `next/font` eliminates layout shift and self-hosts fonts
- Use CSS variables for flexible font usage
- Variable fonts provide all weights in one file
- Configure Tailwind to use font variables
- Use `display: 'swap'` for best user experience
- Group font definitions in a separate file for reusability

## Questions & Answers

### Q: Do I need to download font files for Google Fonts?
**A:** No. `next/font/google` automatically downloads and self-hosts them at build time.

### Q: How do I use a font only on certain pages?
**A:** Import and apply the font in a specific layout or component instead of the root layout.

### Q: Can I use multiple fonts from the same family?
**A:** Yes, specify multiple weights and styles in the configuration.

### Q: What's the performance difference between variable and static fonts?
**A:** Variable fonts are usually smaller when using multiple weights, but larger for single weights.
